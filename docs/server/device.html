Устройство сервера

На данный момент сервер представляет собой мобильное устройство на платформе Android с предустановленным программным обеспечением для корректной работы проекта.

Программное обеспечение содержит основные серверные утилиты для запуска сервера. Для проекта был выбран легкий и быстрый, а главное выполняющий все необходимые задачи проекта, lighttpd сервер. В нем были предустановлены модули для выполнения функций взаимодействия с трутнем, а также обработки полученных данных. Это такие модули как: PHP (новой версии), MySQL (новой версии) и др. PHP - модуль выполняет обработку полученной информации и динамического взаимодействия с сервером, а MySQL для надежного и эффективного хранения полученной информации.

В дальнейшем планируется переход на более мощную серверную платформу нежели мобильная. Также планируется замена серверного ПО на nginx, а так как на nginx могут быть установлены модули (PHP, MySQL и др.) проблем с переходом не будет.
