Сервер

Предыдущее Следующее

Сервер – это оборудование, которое использует серверное программное обеспечение. Он оптимизирован для работы с другими компьютерами (клиентами). Клиентами сервера могут быть компьютеры, телефоны, факсы, принтеры и все другие устройства, которые подключаются к интернету. Чем больше информации Вы планируете хранить на серверном оборудовании, тем мощнее должен быть сервер.

Для исследования в данном проекте используется lighttpd сервер, установленный на мобильной платформе, со всеми необходимыми модулями для проекта ( это и php, mysql и многие другие). В дальнейшем планируется переход на более мощный сервер ( на компьютерной платформе), например nginx.

Относится к части общего проекта beeIT-MADI. Участники проекта и другая информация указаны на главной странице.
