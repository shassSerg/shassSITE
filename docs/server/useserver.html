Обслуживание сервера:

Пояснение

Для предотвращения поломок или возникновения проблем во время эксплуатации сервера следуйте инструкциям ниже.

Для дополнительной информации смотрите главную страницу.

# Период Необходимые действия
1 Каждый месяц Принудительная установка обновлений через WindowsUpdate.
2 Каждый месяц Плановая перезагрузка.
3 Каждый месяц Запланированный полный антивирусный осмотр.
4 Каждый месяц Проверка системы на вирусы с помощью Dr.Web CureIt.
5 Каждый месяц Дефрагментация дисков утилитой MyDefrag.
6 Каждый месяц Проверка создания backup`ов.
7 Каждый месяц Проверка корректного восстановления резервных копий файлов и SQL-баз.
8 Каждый месяц Контроль записей в системном журнале на предмет наличия ошибок. Анализ данных мониторинга.
9 Каждый месяц Контроль мониторинга – корректная работа службы, корректная отправка и доставка уведомлений.
10 Каждые 2 месяца Удаление temp-файлов и старых log-файлов — папки С:\Windows\system32\Logfiles, C:\Documents and settings\%user%.
11 Каждые 2 месяца Проверка открытых портов.
