Бесконтактная док-станция

Предыдущее Следующее

Бесконтактная док-станция - это док-станция для управления дронами и слежения за их состоянием и местоположением. Бесконтактная док-станция позволяет управлять дронами, отдавать им задания, собирать с них информацию и отправлять её на сервер.

В данный момент "бесконтактной док-станцией" является мобильный телефон, на который установлено мобильное приложение BeeIT-MADI, разработанное специально для нашего проекта. Через приложение можно смотреть отчет по каждому заказу, информацию о дроне и о файлах. Постепенно мы будем обновлять и расширять его возможности.

Приложение написано на языке Java для операционной системы Android.
