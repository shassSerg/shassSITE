Обслуживание бесконтактной док-станции:

Пояснение

Для предотвращения поломок или возникновения проблем во время эксплуатации бесконтактной док-станции следуйте инструкциям ниже.

Для дополнительной информации смотрите главную страницу.

# Период Необходимые действия
1 При запуске Проверка соединения.
2 При запуске Проверка соответствия логина и пароля.
3 При запуске Обновление данных.
4 Раз в неделю Проверка наличия обновлений.
5 Раз в неделю Проверка работоспособности.
6 Раз в неделю Чистка кэша телефона.
7 При завершении Выходите из аккаунта для сохранности данных.
